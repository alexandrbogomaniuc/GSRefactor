# Premium Slot Release-Pack Example

This folder keeps one intentionally committed example of generated release-pack metadata.

Canonical behavior:
- Real release-pack outputs are generated under `games/<gameId>/release-packs/<releaseId>/`.
- Those generated outputs are ignored by git.

Included example files:
- `release-packs.index.example.json`
- `RELEASE_PACK_SUMMARY.example.json`
