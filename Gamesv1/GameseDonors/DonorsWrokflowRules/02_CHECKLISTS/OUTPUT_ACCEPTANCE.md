# Output Acceptance

## Raw evidence acceptance

- The run folder contains the required files and asset subfolders.
- Coverage minimums are met with actual artifacts.
- Every observed spin row points to a real evidence filename.
- Sanitization is documented.

## Video acceptance

- Each valid video has visible donor UI and usable thumbnails.
- Invalid video is marked invalid and not counted.
- Factual descriptions match the actual clip content.

## Reconciliation acceptance

- Artifact audit exists.
- Reconciled docs match committed media.
- Replaced artifacts are clearly identified.

## Canonical pack acceptance

- Research evidence sits under the canonical donor folder.
- Factual docs cite evidence.
- Unknowns remain marked `UNOBSERVED`.
- Build handoff guidance does not change GS contracts.

## Optional rare-state acceptance

- Rare-state evidence is captured only if clearly observed.
- If not observed, a negative-result note exists with truthful scope.
