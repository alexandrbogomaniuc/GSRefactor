# /scaffold_pixi — Scaffold PixiJS v8 + TS + Vite

When triggered:
1) Scaffold a PixiJS v8 project using Pixi Create CLI OR a Vite TS template if CLI is unavailable.
2) Ensure dev server runs.
3) Add basic “Boot scene” rendering a sprite and FPS overlay placeholder.
4) Print exact commands to run dev + build.
Acceptance:
- `npm run dev` starts successfully.
- A visible canvas renders.
