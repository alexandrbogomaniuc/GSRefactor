# /perf_audit — Performance Audit (Mobile-first)

When triggered:
1) Identify top likely perf risks (filters, overdraw, texture sizes, batching).
2) Add instrumentation hooks (FPS + frame time spikes).
3) Provide a prioritized optimization list (max 10 items).
Acceptance:
- A clear, prioritized list exists.
- At least 3 concrete code changes are proposed.
