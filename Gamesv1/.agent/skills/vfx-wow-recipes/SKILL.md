# VFX WOW Recipes

## Goal
Generate premium-looking, mobile-safe VFX stacks.

## Instructions
For each requested effect, provide:
1) Visual goal (what player should feel)
2) Layer stack (reels / win fx / UI)
3) Implementation steps
4) Asset requirements (Spine/Effekseer/sprite sequences)
5) Performance risks + fallback

## Constraints
- Avoid full-screen heavy filters in gameplay loops.
- Prefer authored effects over random particles.
