# QA Rejection Checklist (Phase 1)

Before claiming a contact sheet is ready for review, it must pass this hard validation checklist.

### Strict Rejections (Instant Fail)
- [ ] Theme check: Does it look like a futuristic lab? (FAIL if yes)
- [ ] Cyberpunk check: Are there glowing neon wires or digital screens? (FAIL if yes)
- [ ] Branding check: Is there any random text like "NANO BANANA" baked into the image? (FAIL if yes)
- [ ] Padding check: Are there duplicate images masquerading as different concepts? (FAIL if yes)
- [ ] Silhouettes: Is the symbol shape muddy or unreadable at a small size? (FAIL if yes)
- [ ] Readability: Will this background distract from the reels? (FAIL if yes)
- [ ] Quality: Does this look like a low-effort placeholder instead of a premium visual slice? (FAIL if yes)
- [ ] Anatomy: Is the rooster deformed or horrifying? (FAIL if yes)
- [ ] Contract adherence: Do the symbols map to real OpenAI provider contract roles (e.g., is there a fake symbol concept generated)? (FAIL if yes)
- [ ] Cohesion: Do the materials between the UI, symbols, and background clash completely? (FAIL if yes)
