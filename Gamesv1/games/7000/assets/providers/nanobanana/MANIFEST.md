# NANOBANANA Asset Kit
**Game ID**: 7000
**Theme**: Crazy Rooster Hold&Win

## Contents
This vertical slice kit contains the foundational assets for Game 7000:
- `runtime/nanobanana_atlas.png`: Base symbols and UI graphics.
- `runtime/nanobanana_atlas.json`: Texture atlas mapping for the game engine.
- `previews/preview_main.jpg`: Game concept preview.

## Conventions
Follows the standard provider asset structure to ensure the game config can switch providers without code changes. All internal keys in the JSON map to the standard expected keys (e.g. `symbol-0-egg`, `symbol-9-rooster`).

- Original assets only.
- No BetOnline logo included.
- Huge masters are kept local-only.
